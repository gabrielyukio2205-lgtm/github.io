Sobre J.A.D.E.

J.A.D.E. (Just A Digital Entity) é uma assistente de inteligência artificial avançada projetada para auxiliar em tarefas complexas, desde programação e análise de dados até pesquisa acadêmica aprofundada.

Segurança & Privacidade

Interações seguras e processamento de dados focado em privacidade.

Assistência de Código

Geração de código otimizado, debug e explicações técnicas detalhadas sem leitura em voz alta dos blocos.

Scholar Graph

Modo especializado para pesquisas acadêmicas e análise de grafos de conhecimento.

Interface de Voz

Interação natural por voz com feedback visual e controle total de áudio.

Como Usar

  • Chat: Digite suas perguntas ou use o microfone para falar com a J.A.D.E.
  • Imagens: Cole imagens diretamente (Ctrl+V) ou use o botão de anexo para análise visual.
  • Modos: Alterne entre J.A.D.E. (Geral) e Scholar Graph (Acadêmico) na barra lateral.
  • Áudio: Controle a leitura das respostas com os botões de Mute e Parar na interface de input.

Tecnologias

Este projeto utiliza tecnologias de ponta como Groq (LLM), FAISS (Banco Vetorial), Gradio e DuckDuckGo Search, integrados em uma interface "Tron Minimalist" desenvolvida com HTML5, CSS3 e JavaScript puro.
